Ha llegado el momento de que pongas en práctica lo visto en clase. Para hacer esto, ejecute los pasos que se enumeran a continuación.

1) Crea una carpeta (donde prefieras) y dentro de ella guarda el archivo index.html, con el siguiente contenido:

<!DOCTYPE html>
<html lang = "en">
<head>
     <meta charset = "UTF-8">
     <title> Cursos de Alura </title>
</head>
<body>
     <ul>
         <li> Vagrant </li>
         <li> Docker </li>
         <li> Ansible </li>
         <li> Integración continua </li>
     </ul>
</body>
</html>

2) En la terminal (o Git Bash, en Windows) navega hasta la carpeta recién creada (usa el comando cd para navegar entre carpetas);

3) En la carpeta del proyecto, ejecuta el comando git init para inicializar un repositorio Git;

4) Ejecuta el comando git status para asegurarte de que estás en un repositorio de Git y de que el archivo index.html está siendo reconocido. Lee el resultado de este comando con calma. No todo estará claro todavía, ¡pero durante el curso lo va a estar!
